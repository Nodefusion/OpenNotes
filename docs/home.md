# Home

## Introduction

Nodefusion Open Notes.

This repository contains the open notes for Nodefusion. It is built using Docusaurus, a static site generator that makes it easy to create and maintain documentation and blog websites.

All the content is given without any warranty and is for informational purposes only.

Normally, commits and pull requests to this repository are open to the public.

This is one part of how Nodefusion does the Public free / open source and Community engagement and sharing.

Target content is public sharing information about IT technical configuration including in IT architecture design, as well as information about organization/business processes and standard best practices.

[Docusaurus blogging features](https://docusaurus.io/docs/blog) are powered by the [blog plugin](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-blog).

Please continue browsing the [blog posts](/blog/) and searching for your related topics. We will be sharing our thoughts and ideas on a regular basis, so please stay tuned for more updates.

Please use blog posts for sharing information about IT technical configuration including in IT architecture design, as well as information about organization/business processes and standard best practices.
